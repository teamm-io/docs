### Body

<ParamField path="name" type="string" required>
 Name of your application
</ParamField>
<ParamField path="email" type="string" required>
 Email of your application
</ParamField>

Example:
```json
{
"name": "My awesome app";
"email": "awesome.app@mail.com";
}
```


### Response

As a response you will get your APP ID. Please save it for future use.
